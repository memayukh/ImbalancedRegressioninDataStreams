-------------------------------
PhiRelevance for Relevance Calculation
-------------------------------
PhiRelevance module in Python is used to create relevance functions for a target continous variabbles

This module allows to create PhiRelevance Function equivalent to Fortran and C implementation in Python.
C Implementation: https://www.dcc.fc.up.pt/~rpribeiro/uba/
Fortran Implementation: https://github.com/paobranco/UBL/tree/master/src

-------------------------------
PhiRelevance Module Structure
-------------------------------
This module contains 3 files
    1:
    PhiUtils.py - Script loaded with python functions to create Control points and Relevance.
    for documentation and help on this script
        In Command Prompt:
            from PhiRelevance import PhiUtils
            help(PhiUtils) gives the information about callable functions with their prototype and usage.

    2:
    phi.f90 - Fortran script from https://github.com/paobranco/UBL/tree/master/src
    This fortran file is used by PhiUtils.py to create python wrapper for fortran code.

    3:
    readme.txt
    Description and Documentation for PhiRelevance Module.

    4: (optional)

    For Mac and Liux Operating System: phif90_pwrapper.cpython-38-darwin.so
    For Windows Operating System: phif90_pwrapper.cp39-win_amd64.pyd

    When PhiUtils.py is imported from PhiRelevance, it creates a file 'phif90_pwrapper.cpython-38-darwin.so' or 'phif90_pwrapper.cp39-win_amd64.pyd' in the current working directory.
    'phif90_pwrapper.cpython-38-darwin.so/phif90_pwrapper.cp39-win_amd64.pyd' is an intermediate file generated by PhiUtils.py when phi.f90 (Fortran implementation of ultilty) is wrapped with python callable.

    Note: phif90_pwrapper.cpython-38-darwin.so/phif90_pwrapper.cp39-win_amd64.pyd file created in the directory from where PhiRelevance module is imported and used.
    This creating also results in many warning messages - nothing to bother.

-------------------------------
Usage
-------------------------------

Usage 1 for extremes method
-------------------------------
from PhiRelevance import PhiUtils

data = [1.1, 2.2, 1,4, 1.7, 1.8, 4.2, 2.8, 1.9, 3.8]
method = 'extremes'
extrType = 'both'
coef = 0.05

controlPts, npts = PhiUtils.phiControl(data, method = method, extrType = extrType, coef = coef)
yPhi, ydPhi, yddPhi = PhiUtils.phi(data, list(controlPts), npts, method)

# code to plot relevance over continous target variable
import matplotlib.pyplot as plt
plt.plot(data,yPhi,'ro')
plt.xlabel('Continous Target Variable')
plt.ylabel('relevance')
plt.show()

Usage 2 for range method
-------------------------------
from PhiRelevance import PhiUtils

data = [1.1, 2.2, 1,4, 1.7, 1.8, 4.2, 2.8, 1.9, 3.8]
method = 'range'
controlPts = [[1,1,0],[2,1,0],[3,1,1]]

controlPts, npts = PhiUtils.phiControl(data, method = method, controlPts = controlPts)
yPhi, ydPhi, yddPhi = PhiUtils.phi(data, list(controlPts), npts, method)

# code to plot relevance over continous target variable
import matplotlib.pyplot as plt
plt.plot(data,yPhi,'ro')
plt.xlabel('Continous Target Variable')
plt.ylabel('relevance')
plt.show()
